### Warファイルを含むJetty起動用のjarファイルを作成する
Warファイル参考は[HelloJersey](https://github.com/pgtwitter/HelloJersey)
####起動
`java -jar /path/to/JettyWithWar.jar`  

`java -Dport=8080 -jar /path/to/JettyWithWar.jar`
